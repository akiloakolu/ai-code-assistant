# AI Code Assistant

A simple web-based application that allows users to ask coding questions and receive explanations and code examples.

## Features
- Input coding questions
- Generate code snippets
- Provide simple explanations
- Interactive web interface

## Tech Stack
- Python (Flask)
- HTML/CSS
- AI-assisted logic

## Purpose
This project demonstrates the ability to build interactive applications using Python and simulate AI-powered assistance for developers.

## Future Improvements
- Integrate OpenAI API
- Add syntax highlighting
- Improve UI/UX
- Add multi-language support
